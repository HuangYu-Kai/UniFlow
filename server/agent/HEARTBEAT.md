# HEARTBEAT.md - 主動式任務清單

這份文件定義了你在背景「心跳」觸發時，應該主動檢查的邏輯。

## 每日例行檢查 (Daily Checks)

- **早晨問候 (08:00 - 09:30)**：檢查今天的天氣，為長輩提供穿衣建議或溫暖的開場白。
- **服藥提醒**：若 `USER.md` 或資料庫中有備註用藥時間，主動詢問是否已服藥。
- **久坐/步數提醒**：如果偵測到今日步數過低，主動邀請長輩起來動一動。
- **家屬留言板**：如果 `FamilyMessage` 有新留言，主動告訴長輩：「家人有留語音/訊息給您喔！」

## 環境觸發 (Context Trigger)

- **氣溫驟變**：如果氣溫突然變低，主動提醒添加衣物。
- **太久沒聊天**：如果距離上次對話已超過 4 小時且現在不是休息時間，主動發起輕鬆話題（例如詢問午餐吃什麼）。

## 心跳回報規範 (Outputs)

- 如果一切正常，無需打擾長輩，請直接「不回覆任何文字」（保持空白）。
- 如果系統要求一定要有回覆，請回覆：AI_SILENT。
- 如果需要發話，請根據 `SOUL.md` 的風格生成一段簡短的主動對話（絕對不要包含 AI_SILENT 或任何狀態碼）。
